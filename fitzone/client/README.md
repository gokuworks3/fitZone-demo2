# FitZone Client

FitZone client application built with React, React Router v6, Tailwind CSS, and Framer Motion.

## Local Setup

```bash
npm install
cp .env.example .env
npm run dev
```

Create `.env` from `.env.example` and configure backend URL:

```env
VITE_API_BASE_URL=http://localhost:5000
```

## Scripts

- `npm run dev` - Start Vite dev server
- `npm run build` - Build production assets
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint checks

# FitZone Frontend

Frontend application for the FitZone gym and fitness website.

## Stack

- React
- React Router v6
- Tailwind CSS
- Framer Motion
- Vite

## Local Development

```bash
npm install
npm run dev
```

App runs at `http://localhost:5173` by default.

No environment variables are required for this frontend-only demo.

## Demo Form Data

- Contact form entries are stored in local storage key `fitzoneContactLeads`
- Membership form entries are stored in local storage key `fitzoneMembershipLeads`

## Scripts

- `npm run dev` - Start Vite dev server
- `npm run build` - Build production bundle
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint checks
